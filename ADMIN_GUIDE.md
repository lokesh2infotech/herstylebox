# Admin Guide - HerStyleBox

Welcome to the HerStyleBox admin panel! This guide will help you manage your inventory easily.

## Getting Started

1. **Access the Admin Panel**
   - Navigate to `/admin` in your browser
   - Or click "Admin" in the main navigation menu

## Adding a New Product

1. Click the **"Add Product"** button (top right)
2. Fill in the product details:
   - **Product Name** (required): e.g., "Floral Summer Dress"
   - **Description**: A brief description of the product
   - **Price** (required): Enter the price in dollars (e.g., 49.99)
   - **Category**: Optional category (e.g., "Dresses", "Tops", "Bottoms")
   - **Size**: Available sizes (e.g., "S, M, L" or "4, 6, 8")
   - **Color**: Product color (e.g., "Pink", "Blue")
   - **Stock Quantity**: How many items are available
3. **Upload Product Image**:
   - Click the upload area
   - Select an image file from your computer
   - Wait for the upload to complete
   - The image will appear in the preview
4. Click **"Add Product"** to save

## Editing a Product

1. Find the product in the admin table
2. Click the **Edit** icon (pencil) next to the product
3. Modify any fields you want to change
4. Click **"Update Product"** to save changes

## Deleting a Product

1. Find the product in the admin table
2. Click the **Delete** icon (trash can) next to the product
3. Confirm the deletion

## Tips for Best Results

- **Product Images**: Use high-quality images with good lighting. Square images work best (1:1 ratio).
- **Descriptions**: Write clear, appealing descriptions that highlight key features.
- **Pricing**: Be consistent with your pricing format (e.g., always use .99 or .00).
- **Stock Management**: Update stock quantities regularly to keep inventory accurate.
- **Categories**: Use consistent category names to help customers find products easily.

## Managing Inventory

- View all products in the admin table
- See at a glance: product name, price, stock, and category
- Quickly edit or delete products as needed
- The table updates automatically after changes

## Need Help?

If you encounter any issues:
- Make sure all required fields (Name and Price) are filled
- Check that image files are in common formats (JPG, PNG)
- Ensure prices are entered as numbers (e.g., 49.99, not $49.99)

Happy managing! 🎉
